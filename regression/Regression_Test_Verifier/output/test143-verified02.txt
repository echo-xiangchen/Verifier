Formula: ((((not r) and (s or q)) = (p => ((m = (i * j)) and r))) = (((m < i) or (not k)) => (i <= j)))
Where: 
    p : BOOLEAN
    q : BOOLEAN
    r : BOOLEAN
    s : BOOLEAN
    i : INTEGER
    j : INTEGER
    k : BOOLEAN
    m : INTEGER

Is not a tautology. Here is a counter example: 
    r : false
    j : -1
    i : -5
    m : 0
    k : true
    p : false
    q : false
    s : false
