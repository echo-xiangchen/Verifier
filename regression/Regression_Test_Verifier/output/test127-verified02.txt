Formula: (((not r) and (s or q)) = (p => ((m >= (i * j)) and (not (i >= m)))))
Where: 
    p : BOOLEAN = true
    q : BOOLEAN = false
    r : BOOLEAN = true
    s : BOOLEAN = false
    i : INTEGER = 5
    j : INTEGER = 1
    m : INTEGER = 35

Is not a tautology. Here is a counter example: 
    p : true
    r : true
    m : 35
    j : 1
    i : 5
    s : false
    q : false
