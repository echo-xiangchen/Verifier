Formula: (((not r) and (s or q)) = (((p or (m = (i * n))) and (not (i >= m))) => (n > ((m * i) + j))))
Where: 
    p : BOOLEAN = true
    q : BOOLEAN = (not p)
    r : BOOLEAN = ((not p) => (q and p))
    s : BOOLEAN = ((r and p) = (q and p))
    i : INTEGER = ((5 * 6) - 9)
    j : INTEGER = (1 + 9)
    m : INTEGER = (35 - (12 * 2))
    n : INTEGER = ((i + (m * j)) - 25)

Is not a tautology. Here is a counter example: 
    p : true
    r : true
    n : 106
    m : 11
    j : 10
    i : 21
    s : false
    q : false
