Formula: (p = (q and ((i <= j) or (not (m <= 0)))))
Where: 
    p : BOOLEAN = true
    q : BOOLEAN = (not p)
    i : INTEGER = ((5 * 6) - 9)
    j : INTEGER = (1 + 9)
    m : INTEGER = (35 - (12 * 2))

Is not a tautology. Here is a counter example: 
    m : 11
    j : 10
    i : 21
    q : false
    p : true
