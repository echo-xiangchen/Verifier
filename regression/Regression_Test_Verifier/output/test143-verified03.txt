Formula: ((not r0) = (((s1 => ((not l) and (n < (m + (i * j))))) or (i = (j - m))) => (n >= (m - i))))
Where: 
    i : INTEGER
    j : INTEGER
    l : BOOLEAN
    m : INTEGER
    r0 : BOOLEAN
    s1 : BOOLEAN
    n : INTEGER

Is not a tautology. Here is a counter example: 
    m : 0
    i : 0
    j : 0
    n : 0
    r0 : true
    l : true
    s1 : true
