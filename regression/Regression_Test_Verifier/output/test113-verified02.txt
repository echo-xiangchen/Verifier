Formula: (((not r) and (s or q)) = (p => (m > (i * j))))
Where: 
    p : BOOLEAN
    q : BOOLEAN
    r : BOOLEAN
    s : BOOLEAN
    i : INTEGER
    j : INTEGER
    m : INTEGER

Is not a tautology. Here is a counter example: 
    m : 0
    j : 1
    i : 0
    p : true
    q : false
    s : true
    r : false
